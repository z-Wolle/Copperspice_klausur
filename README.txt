In diesem Programm wird ein Slider, eine Progressbar und eine spinbox miteinander verknüpft.
Außerdem gibt es die Möglichkeit über ein "Berechne" Knopf einem die unten in den zwei Spinboxen eingegebenen Zahlen miteinander zu multiplizieren. 
Ich hatte es leider nicht ganz geschafft die LineEdite boxen dafür zu nutzen.

Das Programm wurde erstellt von Daniel Zimmermann.
